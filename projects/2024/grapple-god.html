Grapple God

C++ Unreal Engine 5 GameJam

Overview

Grapple God is a fast-paced action game featuring innovative grappling mechanics and dynamic combat, created during a game jam. Players take on the role of a powerful being who can manipulate gravity and space through advanced grappling abilities.

The game combines fluid movement mechanics with strategic combat, creating an exhilarating experience that rewards both skill and creativity.

Key Features

  • Advanced Grappling System: Physics-based grappling mechanics with realistic rope dynamics
  • Dynamic Combat: Fast-paced action with multiple weapon types and abilities
  • Environmental Interaction: Use the environment to your advantage with the grappling system
  • Procedural Levels: Each playthrough offers unique challenges and layouts

Technical Implementation

This game jam project demonstrates rapid prototyping and efficient development practices, creating a polished experience within tight time constraints.

Technologies Used

  • Engine: Unreal Engine 5.2
  • Language: C++
  • Physics: Chaos Physics Engine
  • Tools: Visual Studio, Unreal Editor, Git

Challenges & Solutions

Time Constraints

Developing a complete game with complex mechanics within the limited timeframe of a game jam.

Solution

Focused on core mechanics first, using Unreal Engine's built-in physics systems as a foundation. Prioritized gameplay over polish, ensuring the core experience was solid before adding additional features.

Physics Performance

Creating smooth grappling mechanics without impacting performance or causing physics glitches.

Solution

Implemented efficient rope physics using Unreal Engine's constraint system and optimized collision detection. Used LOD (Level of Detail) systems to maintain performance during complex physics interactions.
