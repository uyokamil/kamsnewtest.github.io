DishDash

C# Unity GameJam

Overview

DishDash is a chaotic cooking game where players race against time to prepare and serve dishes, featuring multiplayer mechanics and fast-paced gameplay. Created during a game jam, this project demonstrates efficient development practices and engaging multiplayer design.

The game combines frantic cooking mechanics with strategic resource management, creating an exciting experience that's perfect for both solo play and multiplayer sessions.

Key Features

  • Multiplayer Cooking: Real-time multiplayer gameplay with up to 4 players
  • Dynamic Kitchen Management: Manage multiple cooking stations and ingredient supplies
  • Time Pressure Mechanics: Race against the clock to fulfill customer orders
  • Procedural Recipes: Random recipe generation ensures replayability

Technical Implementation

This Unity-based project showcases efficient networking implementation and real-time multiplayer synchronization within the constraints of a game jam.

Technologies Used

  • Engine: Unity 2022.3 LTS
  • Language: C#
  • Networking: Unity Netcode for GameObjects
  • Tools: Visual Studio, Unity Editor, Git

Challenges & Solutions

Multiplayer Synchronization

Ensuring smooth multiplayer gameplay with real-time cooking mechanics and inventory management.

Solution

Implemented Unity's Netcode for GameObjects with custom synchronization for cooking states and inventory. Used authoritative server architecture to prevent cheating and ensure consistent gameplay.

Performance Optimization

Maintaining smooth performance with multiple players, real-time physics, and complex cooking mechanics.

Solution

Optimized asset loading and implemented object pooling for frequently spawned items. Used Unity Profiler to identify bottlenecks and optimize rendering and physics calculations.
