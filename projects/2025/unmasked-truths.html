Unmasked Truths: Shattered Dreams

C++ Unreal Engine 5

Overview

Unmasked Truths: Shattered Dreams is a narrative-driven adventure game that explores themes of identity, reality, and the human condition. Built with cutting-edge Unreal Engine 5 technology, this project pushes the boundaries of storytelling in games.

The game features a unique blend of psychological horror and philosophical exploration, challenging players to question their perceptions of reality while navigating through beautifully crafted environments.

Key Features

  • Advanced Narrative System: Dynamic storytelling that adapts to player choices and actions
  • Unreal Engine 5 Graphics: Utilizing Nanite and Lumen for stunning visual fidelity
  • Psychological Horror Elements: Atmospheric design that creates genuine tension and unease
  • Interactive Environments: Rich, detailed worlds that respond to player interaction

Technical Implementation

This project leverages the full power of Unreal Engine 5, implementing advanced rendering techniques and sophisticated gameplay systems to create an immersive narrative experience.

Technologies Used

  • Engine: Unreal Engine 5.3
  • Language: C++
  • Graphics: Nanite, Lumen, Niagara
  • Tools: Visual Studio, Unreal Editor, Git

Challenges & Solutions

Performance Optimization

Balancing visual fidelity with performance across different hardware configurations while maintaining the game's atmospheric quality.

Solution

Implemented dynamic quality scaling and optimized asset streaming. Used Unreal Engine 5's built-in performance profiling tools to identify bottlenecks and optimize rendering pipelines.

Narrative Complexity

Creating a branching narrative system that maintains coherence while allowing for meaningful player choice and consequence.

Solution

Developed a modular dialogue system using Unreal Engine's Blueprint system, combined with custom C++ classes for complex narrative logic and state management.
